# InsightSQL Agent

An AI-powered SQL Business Intelligence Agent built with Gemini 2.5 Flash, Streamlit, and SQLite.

## Features

- Natural Language → SQL Generation
- SQL Safety Validation
- Editable SQL Before Execution
- SQLite Query Execution
- KPI Dashboard
- Automatic Data Visualization
- AI-generated Business Insights
- CSV Export
- Query History

---

## Tech Stack

- Python
- Streamlit
- SQLite
- Google Gemini API

## Run

Install packages

```bash
pip install -r requirements.txt
```

Run

```bash
streamlit run app.py
```
